Multiple Playlist Parser

The Multiple Playlist Parser can be used to parse the following playlist types on the fly:

  • Native Instruments Traktor nml
  • Windows Media Player, old asx
  • Windows Media Player, current wpl
  • iTunes xml
  • Generic M3U file m3u
  • Playlist file pls

Usage:

Select your playlist from your harddrive. Once selected the parser will immediatelly display your playlist in a form (only Artist and Title will be displayed).
